Revista dos Vegetarianos

Projeto de redesign de revista no ramo alimentício. A proposta era fazer o uso do design para reestruturar um projeto gráfico de uma revista. Com isto, fazer o uso dos conhecimentos em identidade visual para formatar a sua diagramação. Projeto feito no Illustrator, Photoshop e Indesign.
